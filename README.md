# Blood Connect Web

This is a web-based blood donation system designed to connect donors and recipients.

## Features
- User registration
- Donor search by blood group
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project is created to solve real-world problems by helping people find blood donors quickly.

## Future Improvements
- Add location-based search
- Add database integration
- Add login system
